# Credit Scoring Model

## Objective
Predict an individual's creditworthiness using machine learning.

## Algorithms Used
- Random Forest Classifier

## Features Used
- Income
- Loan Amount
- Credit History
- Debt Ratio
- Age

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Project Structure

```text
Credit-Scoring-Model/
│
├── dataset/
├── images/
├── models/
├── notebooks/
├── app/
├── main.py
├── requirements.txt
└── README.md
```

## Run Project

Install requirements:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```
